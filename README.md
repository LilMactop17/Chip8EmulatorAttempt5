# Chip 8 Emulator

## How to Run
*Visual Studio is Required*
1. Clone the Repo
2. Open the project solution file (Chip8EmulatorV2.sln)
3. Build the Project
4. Edit Command Arguments by going to:
     Project -> Properties -> Configuration Properties -> Debugging
5. Paste this in the the Command Arguments: 20 3 Tetris.ch8

20 -> Size of the game screen, 3 -> Speed of the game, Tetris.ch8 -> Game Rom
